Projetos + votação

Fluxo simplificado para as ADMs: crie/edite o projeto, depois configure a votação e as opções do mesmo projeto na mesma tela.

1. Criar/editar projeto

Ou selecione uma imagem do computador. Cards: 1200 x 675 px (16:9).

2. Votação do projeto

Escolha o projeto, abra/feche a votação e adicione as opções. Não precisa mais usar uma aba separada de votação.

3. Adicionar opção de voto

Ou selecione a imagem do computador. Hand Banner: 3:1.

Publicação

Desmarcado: os números ficam visíveis apenas no painel administrativo. Pode ser alterado a qualquer momento sem apagar votos.

Exemplo: se colocar 4, cada pessoa pode escolher até 4 artes na Frase 1, até 4 na Frase 2 e até 4 na Frase 3. Depois de confirmar, não poderá votar novamente.

Recomendado: bloquear por navegador/dispositivo. A opção por IP pode bloquear pessoas diferentes usando a mesma internet.

Use “Não postar ainda” enquanto estiver cadastrando as artes. Quando estiver tudo pronto, mude para “Postar página”.

RELATÓRIOS DO SITE

Estatísticas

Esta área é somente para consulta. Ela não altera votos, projetos ou configurações do site.

Visão geral

Totais consolidados de participação, votos e acessos.

Votos por projeto

“Pessoas” conta cada navegador/dispositivo uma vez dentro de cada votação. “Votos” soma todas as opções escolhidas.

Acessos por página

Cada carregamento de página é contado como um acesso. Visitantes únicos são estimados pelo navegador/dispositivo.

Textos principais do site

Como alterar fotos

As imagens enviadas pelo painel são convertidas para WebP, redimensionadas e comprimidas antes do envio. Após cada upload, o painel mostra o tamanho original e o tamanho final para você confirmar a otimização.
